# spring-boot-angular6-maven-project
Build and package spring boot and angular6 into a deployable war file

### Build

```
mvn package
```

### Run

```
java -jar app-server-0.0.1-SNAPSHOT.jar
```

### Browse

Application
http://localhost:8080

h2 web ui
http://localhost:8080/h2

swagge web ui
http://localhost:8080/swagger-ui.html
